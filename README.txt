CPU Scheduling Simulator Project

Name: NITU
Roll Number: 24MIP10006

Description:
This project is a CPU Scheduling Simulator developed in Python.
It implements different scheduling algorithms to calculate
waiting time, turnaround time, and display Gantt chart.

Algorithms Implemented:

1. First Come First Serve (FCFS)
2. Shortest Job First (SJF)
3. Round Robin (RR)
4. Shortest Remaining Time First (SRTF)

How to Run:

1. Open terminal in project folder
2. Run command: python main.py
3. Enter process details as asked
4. Select the scheduling algorithm

Tools Used:

* Python
* VS Code
